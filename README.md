# make.contrapaul.com — maker & creator portfolio
Bold dark industrial theme. For fans, collaborators, and sponsors.
## To add a project
1. Add a new `.project-card` block in the projects grid in `index.html`
1. Set `data-type` to one of: `print`, `electronics`, `build`
1. Optionally create a `/projects/project-name.html` detail page
## To add a file to the files table
Add a new `.file-row` div inside `.files-table` in `index.html`.
## To update your stats
Replace the `—` placeholders in the collab section with real numbers.
## Structure
- `index.html` — main page
- `style.css` — all styles
- `projects/` — individual project detail pages
- `assets/` — project images
